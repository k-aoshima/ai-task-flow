# AI TaskFlow 利用ガイド

AI TaskFlowは、Google Gemini AIを活用してタスク管理を効率化するChrome拡張機能です。
現在のコンテキスト（開いているWebページ）を理解し、自動的に関連タスクを提案したり、抽象的なタスクを具体的なアクションに分解したりします。

## 🚀 主な機能と使い方

### 1. タスクの作成とAI自動分解
通常のタスク追加だけでなく、AIによる詳細なタスク分解が可能です。

- **手順**:
  1. メイン画面上部の入力エリアにタスクを入力します（例：「プレゼン資料を作成する」）。
  2. **「タスク作成」**ボタンをクリックします。
  3. AIがタスクの内容を解析し、以下を自動設定してリストに追加します：
     - **サブタスクへの分解**: 大きなタスクを手順ごとに分割
     - **優先度**: 重要度・緊急度の自動判定
     - **見積もり時間**: 各作業にかかる時間の予測

### 2. コンテキスト認識 (Context Awareness)
ブラウザで開いているWebページに基づいて、今やるべきタスクをレコメンドします。

- **仕組み**:
  - 例えば `github.com` を開いている場合、GitHubに関連するタスク（「PRレビュー」「Issue対応」など）に**「おすすめ」**バッジが表示されます。
  - 推奨タスクはリストの上位に表示されやすくなります。

### 3. スマート保留タイマー (Interrupt Handling)
急な会議や割り込みが入った際に、タスクを一時保存してリマインダーを設定できます。

- **手順**:
  1. 実行中タスクの**一時停止ボタン（⏸）**をクリックします。
  2. 復帰までの時間を選択します（15分、30分、1時間など）。
  3. 設定時間が経過するとブラウザ通知が表示され、スムーズに作業へ復帰できます。

### 4. 実行中タスクの集中管理
「実行中のタスク」と「全タスクリスト」が分かれており、今集中すべきことにフォーカスできます。

- **機能**:
  - **開始/停止**: プレイボタンでタイマーを開始し、実績時間を計測します。
  - **並び替え**: ドラッグ＆ドロップで優先順位を自由に変更できます。

## ⚙️ 設定とカスタマイズ

### APIキーの設定
本拡張機能は Google Gemini API を使用します。
1. ヘッダーの**設定アイコン**（⚙️）をクリックします。
2. Gemini APIキーを入力して保存します。
   - ※ APIキーはブラウザ内にのみ保存され、外部には送信されません。

### レイアウトの変更
画面の表示項目を使いやすい配置に変更できます。
1. ヘッダーの**カスタマイズアイコン**（✏️）をクリックします。
2. 各セクション（コンテキスト表示、タスク作成、全タスクリストなど）をドラッグして並び替えます。
3. 再度アイコンをクリックして編集モードを終了します。

## 📥 インストール方法 (開発者向け)
1. リポジトリをクローンまたはダウンロードします。
2. `npm install` を実行して依存関係をインストールします。
3. `npm run dev` でローカルサーバーを起動するか、`npm run build` で拡張機能としてビルドします。
4. Chrome拡張機能管理画面 (`chrome://extensions`) で「パッケージ化されていない拡張機能を読み込む」を選択し、`dist` ディレクトリを指定します。
